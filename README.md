##  

## Login

### Login By Password
```shell
omniedge login -u xxx@xxx.com
```

### Login By Secret-Key
You can generate secret-key on omniedge web.

```shell
omniedge login -s xxxxxx
```

## Register
```shell
omniedge register
```

## Join
You can get the target virtual network id from omniedge web.
```shell
omniedge join -n "virtual-network-id" 
```

## Start

```shell
omniedge start
```
